## Acceptance result

M25.3 is accepted after independent reconciliation. Engine main contains a provider-neutral,
evidence-bound extraction worker that consumes M25.2 normalized references and emits only untrusted,
candidate-only artifacts. The accepted implementation supports deterministic recorded-response replay,
immutable prompt/model/candidate-policy identities, bounded provider routes and attempts, M21 evidence
locator validation, prompt-injection warning propagation, secret-like content rejection, immutable
request/response/candidate/receipt evidence, and operator CLI surfaces.

The implementation does not contain a live provider adapter or a credential path. Provider variability
is isolated behind the `ExtractionProvider` protocol, while deterministic post-processing rejects
unsupported candidates, invalid evidence, authority escalation, unbounded output, secret-like data, and
persisted raw source text in request or receipt evidence.
