# M25 Authority Matrix

The machine-readable matrix is `pilot/m25/m25-1-authority-matrix.json`.

Default authority is deny. Technical capability does not create governance authority.

## Role boundary

- ChatGPT operates GitHub-first: investigation, architecture, safe code changes, issues, branches,
  PRs, CI diagnosis, review, expected-head merge after authorisation, reconciliation, and evidence.
- Codex receives only irreducible local work involving a full checkout, terminal, Docker, browser,
  credentials, deployment secrets, or runtime reproduction.
- Daniel is the knowledge and production authority. Item decisions, destructive identity choices,
  Source merge approval, production pointer approval, pilot inventory, and scale authority cannot
  be delegated.

## Permanent M25 exclusions

M25 cannot enable semantic/hybrid production retrieval or production answer serving. These remain
separate authority decisions and cannot be smuggled through an admission envelope.
